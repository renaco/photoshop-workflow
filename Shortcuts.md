PSD Shortcut
=========

### Keyboard shortcuts save us time 


Set select cursor by default

to

Normal Shortcuts:
Open File(s):  
 
<kbd> Ctrl + O </kbd>

<kbd> CMD + O </kbd>


Hidden Extras:  
 
<kbd> Ctrl + H </kbd>

<kbd> CMD + H </kbd>


Cursor:  
 
<kbd> V </kbd>


Slice:  
 

<kbd> C </kbd>

Change to slice: 

<kbd> SHIFT + C </kbd>


Zoom 100%:  
 
<kbd> CTRL + 1 </kbd>

<kbd> CMD + 1 </kbd>


Zoom In:  
 
<kbd> CTRL </kbd> + <kbd> + </kbd> 

<kbd> CMD </kbd> + <kbd> + </kbd>


Zoom Out:  
 
<kbd> CTRL </kbd> + <kbd> - </kbd> 

<kbd> CMD </kbd> + <kbd> - </kbd>


Move Layer:  

<kbd> SPACER + Click </kbd>


Change Canvas Size

<kbd> Alt + CMD </kbd>
 
<kbd> Alt + Ctrl </kbd>


Change Image Size

<kbd> Alt + CMD + i </kbd>
 
<kbd> Alt + Ctrl + i </kbd>


Hidden tools:  
 

<kbd> Tab </kbd>


Hidden others layer
 

<kbd> Alt + Click </kbd> (on eye icon layer)
  



Select size object on layer
 

<kbd> CMD + Click </kbd> (on object layer)


Save for web: 
 
<kbd> Ctrl + Alt + Shift + S </kbd>
<kbd> CMD + Alt + Shift + S </kbd>
